# Metal Arena — المرحلة 1 (سنگل بلير)

لعبة سيارات حربية أركيد بأسلوب Vigilante 8 — هاي النسخة الأساسية: سيارة + فيزياء + تحكم لمس + ساحة.

## التشغيل

1. نزّل **Godot 4.3 أو أحدث** (النسخة العادية Standard، مو .NET) من godotengine.org
2. افتح Godot → Import → اختار ملف `project.godot`
3. اضغط **F5** للتشغيل

> إذا طلع تحذير "created in older version" اضغط OK وكمّل طبيعي.

## التحكم

### على الحاسبة (للتجربة)
| المفتاح | الوظيفة |
|---|---|
| الأسهم يمين/يسار أو A/D | توجيه |
| S أو سهم أسفل | بريك / رجوع |
| مسافة (Space) | درفت |
| الماوس | يمثل اللمس (مفعّل تلقائياً) |

### على الموبايل
- **التسارع تلقائي** — السيارة تمشي بروحها
- **النص الأيسر من الشاشة:** المس أي مكان ويطلعلك جويستيك توجيه
- **يمين:** زر DRIFT وزر BRAKE (البريك مع التوقف = رجوع)
- إذا انقلبت السيارة، تتعدل بروحها بعد ثانية

## ضبط الإحساس (Tuning)

كل القيم بأعلى `scripts/car.gd` — غيّر وجرّب:

| المتغير | التأثير |
|---|---|
| `engine_power` | قوة التسارع |
| `max_speed` | السرعة القصوى (m/s) |
| `steer_strength` | حدّة الانعطاف |
| `grip` | تماسك الإطارات (أقل = ينزلق) |
| `drift_grip` | التماسك وقت الدرفت |
| `spring_strength` / `spring_damping` | قساوة ونطّة التعليق |
| `extra_air_gravity` | سرعة النزول بعد القفزة |

## تصدير APK للأندرويد

> 🔁 **بدون تنصيب أي شي على حاسبتك:** شوف ملف `GITHUB_BUILD.md` — بناء سحابي مجاني عبر GitHub Actions.

1. **Editor → Manage Export Templates** → Download and Install
2. تحتاج على جهازك: **Java 17 (JDK)** + **Android SDK** (command-line tools) — تحدد مساراتهم من Editor → Editor Settings → Export → Android
3. أول مرة: بنفس صفحة الإعدادات اضغط توليد **Debug Keystore**
4. **Project → Export → Add → Android** → Export Project → يطلعلك APK
5. للملتيبلاير لاحقاً: بإعدادات التصدير فعّل صلاحية **Internet**

> أسرع طريقة للتجربة: وصّل موبايلك USB وفعّل USB Debugging، ويطلعلك زر أندرويد فوق يمين بالمحرر — يبني وينصب ويشغل بضغطة وحدة.

## هيكلية الكود (وليش جاهزة للملتيبلاير)

```
scripts/
├── main.gd            ← يبني الساحة والإضاءة ويربط كلشي
├── car.gd             ← فيزياء السيارة (Raycast suspension)
├── chase_camera.gd    ← كاميرا المطاردة
└── touch_controls.gd  ← اللمس + الكيبورد
```

النقطة المهمة: السيارة **ما تقرأ اللمس مباشرة** — تقرأ من كائن `controls` عن طريق ثلاث دوال (`get_steer / is_drifting / is_braking`). يوم نضيف LAN، سيارات اللاعبين الباقين ناخذ مدخلاتهم من الشبكة بنفس الواجهة بدون ما نلمس كود الفيزياء.
